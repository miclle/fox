# Embedded UI Assets

The OpenAPI CLI embeds these offline UI assets:

- `swagger-ui-dist@5.32.5` (`swagger-ui.css`, `swagger-ui-bundle.js`, `swagger-ui-standalone-preset.js`), Apache-2.0
- `@scalar/api-reference@1.55.1` (`scalar-style.css`, `scalar-standalone.js`), MIT
- `redoc@2.5.2` (`redoc.standalone.js`), MIT
